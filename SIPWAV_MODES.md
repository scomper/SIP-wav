# SIP-wav 工作模式与评判规则

> 最后更新：2026-06-18

## 模式 A — 波形异常检测（独立文件分析） ✅ 已实现

> 命令：`sipwav task --dir ./录音/` 或 `sipwav` 交互模式选 1

不需要参考样本，逐文件快速筛查通话录音的硬伤。

| 检测项 | 规则 | 阈值 | 输出 |
|--------|------|------|------|
| 静音超长 | VAD 连续非语音帧 | > **2s**（`--silence` 可配） | 静音段数 + 最长段位置 |
| 通话截断 | 尾部 5 帧能量梯度 < -3σ 且 RMS→0 | 梯度标准差 ≥3 | 截断位置 |
| 空录音 | RMS 均值低且变化率趋零 | RMS < 0.01, 变化率 < 0.05 | "空录音" |
| 纯音/忙音 | 频谱主频能量占比 > 60% | 占比 > 0.6 | 主频 Hz |
| 能量异常 | RMS 变化率异常低或高 | 变化率 < 0.05 | "能量异常" |

**输出格式**：
```
  ⚠️  不正常的.wav    1513s  静音 / 截断 | >2s 静音 9 段 · 最长 30.6s (251s-)
  ⚠️  持续嘟声.wav     960s  纯音
```

## 模式 B — 样本锚定比对（WAV 锚定） ✅ 已实现

> 命令：`sipwav task --sample 正常.wav --dir ./录音/` 或交互模式选 2

拿一条正常录音做锚，比对待检文件的波形一致性。

**预过滤**：有参考样本时，自动排除特征不匹配的文件：
- 时长比 <0.5 或 >2.0
- 采样率不同（8k vs 16k）
- 能量差异 >50 倍

| 检测项 | 规则 | 阈值 | 输出 |
|--------|------|------|------|
| 时长偏移 | 待检/参考 时长比 | < **0.7**（截断）或 > **1.3**（拖长） | 实际比值 |
| 包络不匹配 | 能量包络余弦相似度 | < **0.6** | 余弦相似度 |
| DTW 漂移 | MFCC 对齐路径偏离对角线 | > **0.3** 且 DTW 代价 > 0.01 | 漂移量 + 代价 |
| VAD 节奏偏移 | 静音段数量差异或位置偏移 | 数量差 > **20%** 或位置偏移 > **0.5s** | 参考/待检 静音段数 |

**输出格式**：
```
  ⚠️  number.wav    600s  过短 / 包络不匹配 / 时间漂移 / 波形不似 | 时长比 0.40, 包络 0.01, DTW代价 176.08
```

**限制**：L1 已判异常的文件不进 L2（省算力）。

## 模式 C — ASR 内容检测（文本→TTS→锚定） ✅ 已实现

> 命令：`sipwav task --sample ref.wav --asr --dir ./录音/` 或交互模式选 3

最精准的模式。从文本生成参考语音，锚定比对内容和播放质量。

| 检测项 | 规则 | 阈值 | 输出 |
|--------|------|------|------|
| 吞字/缺字 | ASR 文本 diff（参考有×待检无） | 匹配率 < **1.0** | 缺失内容 |
| 多余内容 | ASR 文本 diff（待检有×参考无） | 匹配率 < **1.0** | 多余内容 |
| 内容不匹配 | 整体文本相似度 | < **50%** | 相似度值 |
| 无声/噪音 | ASR 返回空 | 无文本 | "无语音" |
| 时间轴漂移 | 数字时间轴对齐后逐个比对 | > **1.0s** | 漂移位置 + 偏差 |

**数字时间轴漂移检测**：
1. ASR 识别出数字序列
2. 找第一个相同数字作为锚点，计算全局偏移
3. 逐个比对后续数字的时间位置
4. 偏差 >1s → 报告漂移

**ASR 控制**：
- 最大文件数：100（超过直接跳过，不执行 ASR）
- 抽样量：10（候选 >10 时均匀抽样，最多分析 100 个）
- 长录音切片：>180s 自动在静音点切片，分段 ASR 后合并
- 交互模式：发现第一个异常暂停，等待用户决定

**输出格式**：
```
  # 候选 ≤10：直接分析
  L3 ASR 分析: 8 文件... 完成 / 2m30s

  # 候选 11~100：均匀抽样 10 个
  L3 ASR 分析: 抽样 10/45 文件... 完成 / 45s

  # 候选 >100：跳过
  L3 ASR 分析: 候选 150 文件，超过上限 100，跳过
```

## 模式 D — 统计基线模式（多样本学习） ⏳ 待实现

> 建议命令：`sipwav learn --normal-dir ./正常录音/ --model model.pkl`
>          `sipwav detect --model model.pkl --dir ./待检/`

适用于人工座席、内容不固定的录音。从大量正常样本学习特征分布。

| 检测项 | 规则 | 阈值 |
|--------|------|------|
| 特征偏离 | 多特征马氏距离 | > 95 分位 |
| 时长异常 | 相对于样本池的时长分布 | < 5% 或 > 95% 分位 |
| 节奏异常 | 说话/静音比例分布 | 偏离均值 3σ |

## 模式 E — 语速/节拍异常 ⏳ 待实现

> 适用：报数录音、固定语速场景

| 检测项 | 规则 | 阈值 |
|--------|------|------|
| 语速过快 | 平均音节时长 < 参考 70% | 短于 0.7× 参考 |
| 语速过慢 | 平均音节时长 > 参考 130% | 长于 1.3× 参考 |
| 卡顿 | 单段内多次停顿 | 同段 VAD 碎片 > 5 段 |

## 模式 F — 通道/方向检测 ⏳ 待实现

## 阈值可配置参数

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `--silence` | 2.0 | 静音报警阈值（秒） |
| `silence_threshold` | 2.0 | 静音检测阈值 |
| `duration_short` | 0.7 | 时长过短比值 |
| `duration_long` | 1.3 | 时长过长比值 |
| `envelope_similarity` | 0.6 | 包络余弦相似度阈值 |
| `dtw_drift` | 0.3 | DTW 漂移阈值 |
| `dtw_cost` | 0.01 | DTW 代价（低于此忽略漂移） |
| `asr_similarity` | 0.5 | ASR 文本相似度阈值 |
| `drift_threshold` | 1.0 | 数字时间轴漂移阈值（秒） |
| `max_asr_files` | 100 | ASR 最大文件数 |
| `asr_sample_size` | 10 | ASR 抽样数量 |
